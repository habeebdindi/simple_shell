A simple shell project
======================

This simple shell project is intended to perform similar operations as the unix shell.

## Output

The output of this simple shell wil be exactly the same as that of *sh*. The only slight difference will be in the error output.The only difference is that when an error is printed, the name of the program is going to be equivalent to the agrv[0] supplied to the simple shell prompt.

## Compilation

The files are compiled using ** gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh **

## Working Modes

The simple shell works in both interactive mode and non-interactive modes, see below

## Language

This simple shell was written in C.
